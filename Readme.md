## Reflection em Java
## Sobre o projeto

Este projeto foi desenvolvido como exercício prático para demonstrar o uso da API Reflection em Java.

O programa cria uma lista dinamicamente utilizando Reflection, adiciona elementos por meio da invocação do método add() e percorre todos os itens da lista também utilizando Reflection, exibindo os valores no console.

## Tecnologias utilizadas
Java
API Reflection
## Funcionalidades
Instancia uma ArrayList via Reflection.
Adiciona objetos à lista utilizando o método add() por Reflection.
Percorre a lista utilizando os métodos size() e get() por Reflection.
Exibe todos os elementos no console.

## Objetivo

Praticar os conceitos de Reflection em Java, compreendendo como instanciar classes, invocar métodos dinamicamente e manipular objetos em tempo de execução, conforme proposto na atividade da EBAC.
